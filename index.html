Delta PACS

Delta PACS es un sistema de archivo y distribución de imágenes médicas (PACS), proporciona un almacenamiento económico y acceso a las imágenes de múltiples modalidades como tomografía computarizada, resonancia magnética, rayos X, ecografía, etc.

¿Cómo funciona Delta PACS?

Usted accede al PACS para ver las imágenes desde cualquier computadora que posea un navegador ( Chrome, Firefox ) dentro o fuera de su institución.
Cualquier computadora se convierte en una estación de trabajo.

Visor de Imágenes para distintas resoluciones y equipos... Computadoras, Notebook y Celulares.

Multiples visores de imágenes. Se pueden conectar a distintos visores de imagenes.
